Functions & Models

The main character of this class is the function. This entire course could be described as a study of progressively more complicated examples of functions. Before diving into that though, let’s start with a general description of a function.

A function is an assignment to every input in some set of inputs a unique output in some set of outputs.

Example: Consider a function that takes as its input a number considered as a temperature measured in °C and assigns it the output that is the same temperature measured in °F. Let’s call the function \(f\) for function.

At its core, all a function is is this input/output correspondence; a function is the set of all pairs \((\text{input}, \text{output}).\) We can express this less illustritavely than above by putting these pairs in a table.
